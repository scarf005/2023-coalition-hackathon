# RT4M: Read The 42 Manual

42 카뎃을 위한 위키 및 알림 슬랙봇

## 실행 방법

```sh
git clone https://github.com/scarf005/2023-coalition-hackathon
cd 2023-coalition-hackathon
npm install
npm run dev
```

루트 디렉토리에 다음과 같이 `.env` 파일을 만들어주세요.

```
SLACK_BOT_TOKEN
SLACK_SIGNING_SECRET
SLACK_APP_TOKEN
PORT
```
